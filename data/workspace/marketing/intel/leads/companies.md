### Netdata
- **GitHub:** [netdata/netdata](https://github.com/netdata/netdata) — 79k stars
- **Descripción:** "The fastest path to AI-powered full stack observability, even for lean teams."
- **Tecnología:** Go, React, Prometheus metrics, eBPF

- **Vacancies:** [careers page](https://netdata.cloud/careers)

### SigNoz
- **GitHub:** [SigNoz/signoz](https://github.com/SigNoz/signoz) — 27k stars
- **Descripción:** Open-source observability platform native to OpenTelemetry (logs, traces, metrics)
- **Tecnología:** ClickHouse backend, Golang, React
- **Posicionamiento:** Alternative a Datadog/NewRelic para empresas que quieren control total de datos
- **Vacancies:** [careers page](https://signoz.io/careers)
- **Comentarios Mauro:** Media prioridad — tema observabilidad, pero más APM tradicional que DevOps enterprise.

### Langfuse
- **GitHub:** [langfuse/langfuse](https://github.com/langfuse/langfuse) — 28k stars
- **Descripción:** Open source LLM engineering platform: LLM Observability, metrics, evals, prompt management, playground, datasets. Integrates with OpenTelemetry, Langchain, OpenAI SDK.
- **Tecnología:** TypeScript, NextJS, PostgreSQL/ClickHouse
- **Posicionamiento:** RAG/agentes IA observability — exactamente el gap que Mauro llena.
- **Comentarios Mauro:** **ALTA PRIORIDAD** — tema "IA aplicada operacional" y "RAG corporativo". Revisar integraciones.

### Kong
- **GitHub:** [Kong/kong](https://github.com/Kong/kong) — 43k stars
- **Descripción:** "🦍 The API and AI Gateway"
- **Tecnología:** NGINX + Lua, OpenResty
- **Posicionamiento:** API Gateway con IA integrada (posiblemente rate limiting, routing inteligente)
- **Comentarios Mauro:** Alta relevancia — DevOps enterprise, gateway layer es parte crítica de arquitectura.

### Dokploy
- **GitHub:** [Dokploy/dokploy](https://github.com/Dokploy/dokploy) — 34k stars
- **Descripción:** Open Source Alternative to Vercel, Netlify and Heroku.
- **Tecnología:** Docker, Portainer UI, Node.js backend
- **Posicionamiento:** PaaS interno para startups que no quieren AWS manual pero necesitan más que Heroku
- **Comentarios Mauro:** Media prioridad — tema "plataforma interna" en el radar de Mauro.
